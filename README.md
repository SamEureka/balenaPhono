## balenaPhono 
### A turntable phono/AUX/USB network streamer

#### Device Variables
| Variable | Example Value | Note |
|---|---|---|
| CHECK_CONN_FREQ | 120 | This is the wifi-connect polling wait time in seconds. The default is 120 seconds, I usually set mine to 3000.  |
